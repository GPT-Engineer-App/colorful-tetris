# colorful-tetris

A Tetris game project, featuring classic Tetris gameplay with moving, rotating, and dropping Tetriminos to clear lines. The visual style should be modern yet reminiscent of the classic game, with bright colors for the Tetriminos and a simple, intuitive interface for controls.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/colorful-tetris.git
cd colorful-tetris
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
